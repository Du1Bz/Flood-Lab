# Flood Lab

『Halo Infinite』に特化したローカル優先の分析・実験プラットフォーム。

## 目標

- 『Halo Infinite』のマッチデータを調査する
- パフォーマンスの傾向を分析する
- セッション別／週次／月次のアクティビティを確認する
- メトリクスや可視化手法を実験する
- 将来の分析に備えて、豊富な生マッチデータを保存する

## 理念

- 生データは不変である
- 分析用データセットは個別に正規化する
- 可視化がストレージの設計を左右してはならない
- 分析を第一に、UIは第二に

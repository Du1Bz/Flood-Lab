# 📖 Flood-Lab 使用方法ガイド

Flood-Lab は Halo Infinite の戦績を可視化・分析するための Streamlit アプリケーションです。  
起動するとブラウザ上で以下の5ページが利用できます。

---

## 📑 ページ一覧

| # | ページ | 役割 |
|---|--------|------|
| 1 | 🌊 **ホーム** | ダッシュボード。サマリー・カレンダーヒートマップ・eMMR推移・インサイトを一目で確認 |
| 2 | 📊 **分析** | フィルターで自由に絞り込んでグラフを探索。トレンド・マップ/ルール・TrueSkill2・オブジェクト |
| 3 | 📅 **レポート** | 週・月・セッション単位の定型レポート。前期間比較付き |
| 4 | 🔬 **仮説検証** | 各種統計検定（t検定・ANOVA・相関・回帰） |
| 5 | 📋 **試合履歴** | 全試合一覧。行クリックで詳細展開。CSVエクスポート機能付き |

## 1. 🌊 ホーム（デフォルトページ）

起動時に最初に表示されるページ。現在の状態を俯瞰する。

### 表示内容

| セクション | 内容 |
|-----------|------|
| サマリーカード | 総試合数・勝率・平均K/D・平均インパクトスコア（ランクのみ）・全試合数 |
| カレンダーヒートマップ | 直近8週間の日別インパクトスコアと試合数 |
| eMMR v2 推移 | 直近50試合の推定MMR推移。区分別に色分け表示 |
| インサイト | 直近30試合から自動検出された傾向（スランプ・連敗・苦手マップ等） |

### 使い方

- サイドバーの「データ更新」ボタンで最新データに更新できます
- インサイトは自動生成。レベル別に色分け表示（ℹ️=情報 / ⚠️=注意 / 🚨=警告）

## 2. 📊 分析

フィルターを自由に設定してデータを探索するページ。4つのタブで構成。

### サイドバーフィルター

| フィルター | オプション |
|-----------|-----------|
| 期間 | 直近N戦（10〜500） / 過去7日 / 過去30日 / カスタム日付 |
| 区分 | すべて / ランクアリーナ / ランクスレイヤー / 等 |
| マップ | すべて / 個別マップ |
| ルール | すべて / Slayer / CTF / Oddball / 等 |

## 3. 📅 レポート

期間を固定してスナップショットとして確認するページ。

### 期間選択

| モード | 内容 |
|--------|------|
| 週 | 過去26週から選択。前週との比較あり |
| 月 | 過去12ヶ月から選択。前月との比較あり |
| セッション | 個別セッションを選択 |
| カスタム | 任意の日付範囲 |

### 表示内容

| セクション | 内容 |
|-----------|------|
| サマリーカード | 試合数・勝率・K/D・命中率・CSR増減合計 |
| eMMR v2 推移 | 期間内のeMMR推移グラフ |
| マップ×ルール勝率HM | 期間内のマップ×ルール勝率ヒートマップ |
| 注目試合 | 最高/最低KDA試合の詳細 |
| 区分別勝率 | 区分ごとの試合数・勝率・KD・CSR増減 |
| 前期間比較 | 勝率・KDA・命中率・KPMの前期間との差分 |

## 4. 🔬 仮説検証

統計的な検定・分析を行うページ。データドリブンな仮説検証が可能。

### 機能一覧

| 分析 | 内容 |
|------|------|
| t検定（勝敗比較） | 勝ち試合と負け試合で各指標に有意差があるか検定 |
| t検定（二群比較） | 任意の二群（ソロvsパーティ・マップA vs B 等）を比較 |
| ANOVA（ルール別比較） | ルール間で指標に有意差があるかを分析 |
| 相関分析 | 2指標間の相関係数・散布図＋回帰直線 |
| 単回帰分析 | 目的変数・説明変数を指定して回帰分析 |
| 勝率シミュレーション | 勝敗を確率変数としたモンテカルロシミュレーション |

### 使い方の例

- 「ソロとパーティでKDAに差があるか？」→ t検定（二群比較）
- 「マップによって命中率は変わるか？」→ ANOVA
- 「KDAとd_rpiの相関は？」→ 相関分析

## 5. 📋 試合履歴

全試合を時系列で確認できる一覧ビュー。

### 機能

| 機能 | 説明 |
|------|------|
| フィルター | 期間・区分・マップ・ルール・勝敗・除外フラグで絞り込み |
| 行クリック詳細 | 任意の試合行をクリックすると全フィールドをポップアップ表示 |
| CSVエクスポート | 現在のフィルター条件で表示中の試合をCSVとしてダウンロード |
| 除外フラグ試合も表示 | 除外フラグ（short_match / incomplete等）の試合も確認可能 |

### フィルターオプション

- 期間: すべて / 今週 / 今月 / 直近N戦 / カスタム
- 除外フラグ: 通常のみ / 除外のみ / すべて

## 💡 活用シナリオ

### シナリオ1: 今日の調子をチェック
1. ホームを開く → カレンダーHMで今日のインパクトを確認
2. インサイトで自動検出された問題をチェック
3. 気になる点があれば分析ページで詳細を掘る

### シナリオ2: 特定のマップ・ルールの苦手分析
1. 分析ページでフィルターを設定（例: Lattice / Oddball）
2. トレンドタブでKDAの推移を確認
3. TrueSkill2タブでK-RPI/D-RPI散布図を見る
4. オブジェクトタブでボールスタッツを確認

### シナリオ3: 週単位の振り返り
1. レポートページで「週」を選択
2. 前期間比較でどれだけ成長したか確認
3. 注目試合で最高・最低試合を振り返る

### シナリオ4: AIにコーチングを依頼
1. 分析ページで気になる期間にフィルター
2. 「📥 JSONを生成」→ ダウンロード
3. AI（Claude等）にJSONをアップロードして分析・アドバイスを依頼

## 📊 各指標の見方

各指標の詳しい説明は [`docs/METRICS.md`](METRICS.md) を参照してください。

| 略語 | 正式名 | 意味 |
|------|--------|------|
| K/D | キル/デス比 | 1.0が均衡。高いほどキル効率が良い |
| KDA | キル-デス+アシスト/3 | 総合貢献度。プラスが望ましい |
| K-RPI | キル相対パフォーマンス | 1.0=期待通り、>1.0=期待超え |
| D-RPI | デス相対パフォーマンス | 1.0=期待通り、>1.0=期待より死ななかった |
| eMMR v2 | 推定MMR | カルマンフィルタ平滑化済み推定実力値 |
| LGAI | ロビー格差 | 正=格上相手、負=格下相手 |
| PWコントロール率 | パワーウェポン支配率 | 0.5が均衡。自チームのPWキル比率 |

## ⚠️ データの注意点

- **OpenSpartan 1.0.10 以下を使用してください。** 1.0.11/1.0.12では一部データが正しく取得できません。
- **モーショントラッカーはRankedでは無効**です。音・視界・コールアウトが主な情報源になります。
- **PWコントロール率はショックライフルが未計上**のため、ショックライフルマップ（Recharge / Solitude / Vacancy）では過小評価になります。
- **除外フラグ**が立った試合（short_match / incomplete等）は集計・グラフから自動的に除外されますが、試合履歴ページでは確認できます。
